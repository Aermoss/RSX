# RSharp
An interpreted programming language.
![rsharp logo](./icon.png)

# How to use
## How to interprete the program
```python main.py --run main.rsx```

## How to transpile the program
### Python
```python main.py --transpile-python main.rsx```

# Exapmles
## Hello, World!
```c++
include "rsxio" : *;

int main() {
    std::rout("Hello, World!");
    return 0;
}
```
